正常实现

1. 求开方

题目描述

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

题解

题目简单，但有两个坑点：

  • 对于 x = 8，它的开方是 2.82842…，最后应该返回 2 而不是 3。
    • 在循环条件为 l <= h 并且循环退出时，h 总是比 l 小 1，也就是说 h = 2，l = 3，因此最后的返回值应该为 h 而不是 l。
  • 边界值
    • 如果直接 m = (l + h) / 2的话，会爆。
    • 因此最好写成 l + (h - l) / 2。

3. 有序数组的 Single Element

题目描述

给定一个只包含整数的有序数组，每个元素都会出现两次，唯有一个数只会出现一次，找出这个数。

题解

  • 由于数组个数一定为奇数，且从偶数0开始，故正常情况下<偶数位>-<奇数位>是相等的。
  • 因此，保证l,m,h都在偶数位，只需判断m当前位与下一位是否相同。
    • 若相同，则查找 (m+2, h)
    • 若不同，则查找 (l, m)
  • 注意，由于第二种情况直接令 h = m，故循环条件应改为 l < h，否则可能会死循环。

4. 第一个错误的版本

题目描述

给定一个元素 n 代表有 [1, 2, …, n] 版本，在第 x 位置开始出现错误版本，导致后面的版本都错误。可以调用 isBadVersion(int x) 知道某个版本是否错误，要求找到第一个错误的版本。

题解

  • 如果第 m 个版本出错，则表示第一个错误的版本在 [l, m] 之间，令 h = m；否则第一个错误的版本在 [m + 1, h] 之间，令 l = m + 1。
  • 因为 h 的赋值表达式为 h = m，因此循环条件为 l < h。

6. 查找区间

题目描述

给定一个按照升序排列的整数数组 nums，和一个目标值 target。找出给定目标值在数组中的开始位置和结束位置。

题解

可以用二分查找找出第一个位置和最后一个位置，但是寻找的方法有所不同，需要实现两个二分查找。

  • 也可以转化一下，我们将寻找 target 最后一个位置，转换成寻找 target+1 第一个位置，再往前移动一个位置。这样我们只需要实现一个二分查找代码即可。
  • 不过这里有个问题就是h的初始值，若为n-1，则在查最后一个位置时会出问题。
